INTRODUCCIÓ JAVASCRIPT

Javascript és un llenguatge de programació interpretat orientat a objectes , amb sintaxis de molt semblant a C però amb metodologies molt semblants la Java . El llenguatge va ser desenvolupat per l’equip de desenvolupament de Netscape als anys 90. Es un llenguatge de programació es interpretat en la part navegador del client però també es pot interpretar en la part de servidor . Permet convinat amb HTML poder fer web dinàmiques amb interacció entre la pròpia web i l’usuari per exemple per realitzar comprovacions o representacions de dades, con-sultat a base de dades ....

Per tal d’organitzar i facilitar la tascar d’actualització i manteniment el mes recomanable es separar en diferents fitxer el codi de programació (JavaScript) i el full d’estil (Css ) i el contingut de la pròpia web (HTML)

El codi JavaScript pot estar inclòs en el mateix fitxer que HTML entre etiquetes <script> Aquí estarà el codi JavaScript integrat dins de HTML </script>

També es pot vincular a la part de la capçalera el fitxer on estarà el codi o les diferents funcions de JavaScript en un fitxer apart <script type="text/javascript" src="../js/funciones.js">; < /script>
