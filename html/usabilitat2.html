Valoració de la Usabilitat: Ítems per a valorar la Usabilitat

Navegació:

  1. S'identifica clarament la situació actual dins de l'espai web
  2. L'enllaç a la pàgina principal s'identifica clarament
  3. Les parts més importants de la web són accessibles des de la pàgina principal
  4. El mapa de la web no presenta una web gran i complexa
  5. Hi ha vinculació profunda
  6. No apareix pàgina de benvinguda prèvia a la principal
  7. Es proporciona una funció de cerca senzilla

Funcinalitat:

  1. La web és còmoda tant per a principiants com para experts
  2. La funcionalitat està etiquetada clarament
  3. Les funcions essencials són accessibles sense deixar la web
  4. Els plug-ins són solament utilitzats per afegir valor

Control de l'usuari:

  1. L'espai reflecteix el flux de treball de l'usuari
  2. L'usuari pot cancel·lar qualsevol acció
  3. Totes les pàgines disposen d'un punt de sortida
  4. La grandària de cadascuna de les pàgines és menor de 50 KB
  5. L'espai és suportat pels principals navegadors
