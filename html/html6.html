ELEMENTS QUE DONEN FORMAT AL TEXT (strong, em, abbr, q…)

Per ésser absolutament amb la utilització del llenguatge HTML, trobem tot un seguit d'etiquetes que fan referència a la importància que té cadacuna de les parts del text.

En primer terme, hi ha dues etiquetes que serveixen per remarcar parts concretes del text; però visualment apareixeran diferent l'etiqueta <strong> marcarà el text en negreta; mentre que l'etiqueta <em> emfatitzarà el text en cursiva.

Una altra família d'etiquetes seran les que utilitzarem per fer algun tipus de citació del text. Entre aquestes hi trobarem les etiquetes <q>, <blockquote> i <cite>. Les diferències com en l'anterior cas, també seran visuals, doncs si fem una citació mitjançant <q> el text es veura entre cometes, amb <blockquote> el text citat apareixerà separat de la resta amb una tabulació a la dreta, mentre que amb <cite> aquest es visualitzarà en cursiva.

Finalment, i per acabar amb aquest apartat, trobem un seguit d'etiquetes amb els que podrem entre altres coses: determinar una data o un moment concret del text <time>, mostrar una abreviatura <abbr>, subíndexs <sub> i superíndexs <sup>, mostrar informació sobre l'autor/a del document HTML <address>, ensenyar el text exactament com ha estat escrit a l'editor d'HTML <pre>, marcar el codi de programació amb <code>, escriure funcions de JavaScript amb <var> entre altres.
