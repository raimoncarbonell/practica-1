VÍDEO

Acabem l'apartat de l'HTML amb el vídeo, un element que se'ns dubte donarà un extra de qualitat i que farà destacar la nostra web per sobre les altres. Habitualment, tots els navegadors web són capaços de llegit perfectament els MPG4s o H.264 que són l'estàndar de compressió en els formats vídeos per a web.

Així mateix, i per major seguretat, és comú utilitzar uns altres dos formats de compressió amb els quals obtindrem menor qualitat, però que ens permetran assegurar-nos que els vídeos es carreguin correctament a la nostra pàgina; ens estem referint als formats .webm i .ogg

A continuació podem veure com carregarem aquest tipus d'arxiu:

<video id = "marc" width = "600" loop poster = "img/marc.png" controls>
<source src = "videos/Marc_mosca_etalonat.mp4" type = "video/mp4">
<source src = "videos/Marc_mosca_etalonat.webm" type = "video/webm">
<source src = "videos/Marc_mosca_etalonat.ogv" type = "video/ogg">

<track src = "mosca.ca.vtt" kind="subtitles" srclang = "ca">
</video>

A més, la línea prèvia al tancament del vídeo, ens serveix per incrustar-hi uns subtítols; aquests s'hauran d'escriure en un arxiu amb extensió .vtt i independent a l'HTML seguint el següent format:

WEBVTT

1
00:00:00.000 --> 00:00:01:000
Res

2
00:00:01:000 --> 00:00:02:000
Encara res!!

3
00:00:01:000 --> 00:00:04:000
Quina nit més feixuga...
(...)
