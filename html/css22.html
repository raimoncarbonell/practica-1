CSS

Els tamanys

Els tamanys es poden definir de dues maneres diferents: amb els pixels (px) o amb percentatge (%). En el cas del text es poden definir les declaracions amb unitats respecte el tamany de la lletra (em). Exemples: 
h1 { font-size: 16px; }
h1 { font-size: 200%; }
h1 { font-size: 2em; }

El model de caixes

El model de caixes és una de les característiques més importants del llenguatge de la fulla d'estils CSS, ja que condiociona el diseny de totes les pàgines web. El model de caixes és el comportament de CSS que fa que totsa els elements de les pàgines es representin mitjançant caixes rectangulars.

Les caixes d'una pàgina es creen automàticament. Cada vegada que s'insereix una etiqueta HTML, es crea una caixa nova rectangular que tanca els continguts d'aquests element.

Així amb els elements: content, padding, border, background image, background color i margin, podem canviar l'aspecte de totes les caixes
