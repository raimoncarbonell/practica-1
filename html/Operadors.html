OPERADORS: ARITMÈTICS, LÒGICS, ASSIGNACIÓ I RELACIONALS

L'operador d’assignació es seu signe es igual ‘=’ . Aquest operador s’utilitza per assignar el valor a una variable o per fer-ne una copia de valor de l’expressió que queda a la dreta del operador en el costat de l’esquera del operador . Els operador operadors aritmètics modifiquen el valor al que s'apliquen i són:

Els operadors relacionals són:

Operador Símbol Acciò
Increment ++ Incrementa una unitat
Disminució -- Disminució
Menor unitari - Torna negatiu un número

Els operadors unitaris matemàtics poden col·locar-se abans (prefixos) o després (sufixos) de l'operando i el seu valor varia segons aquesta posició, ja que l'operador prefixo modifica l'operando abans d'utilitzar el seu valor, mentre l'operador sufix modifica l'operando després d'haver utilitzat el valor.Exemple

x=10 x=10
y=x++; y=++x;
per el que y=10 yx=11 per el que y=11 y x=11

L'operador relacional

L'operador relacional entenem la relació que té un valor respecte a un altre. Es basen en el concepte de vertader o fals i en tots els casos operen amb només dos estats diversos (0/1, encesa / apagada, cert / fals).

Operador Acción
> Major que
>= Major o igual
< Menor que
<= Menor o igual
== Igual
!= Diferent

Els operadors binaris matemátics

Operador Símbol Acció
Suma + Suma dos operands
Resta - Resti el segon operant del primer
Multiplicacio * Multiplica els dos operands
Divisió / Divideix el primer operant pel segon
Resto (módulo) % Facilita la resta de la divisió entre dos operands

Operadors lògics

Els operadors lògics s'utilitzen per prendre decisions sobre les instruccions que heu d'executar la programa en funció de certes condicions El resultat de qualsevol operació que utilitzi operadors lògics sempre és un valor lògic o booleà.

Negació

S'utilitza per obtenir el valor contrari al valor de la variable. La negació lògica s'obté prefixant el símbol "!" l'identificador de la variable

variable !variable
True False
False True

AND

L'operació lògica AND obté el seu resultat combinant dos valors booleans. L'operador s'indica mitjançant el símbol && i el seu resultat només és true si els dos operands són true:

variable1 variable2 variable1 && variable2
true true true
true false false
false true false
false false false

OR

L'operació lògica OR també combina dos valors booleans. L'operador s'indica mitjançant el símbol || i el seu resultat és true si algun dels dos operands és true:

variable1 variable2 variable1 || variable2
true true true
true false true
false true true
false false false
