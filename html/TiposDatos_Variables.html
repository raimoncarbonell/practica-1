TIPUS DE DADES I VARIABLES

Una variable es declara amb la paraula clau var. Exemple var numero.En la declaració no figura què tipus de variable és (per exemple si és text tipus String o si és numèrica tipus Number). Javascript detecta que conté un valor numèric, la considerarà tipus Number. Si conté un valor de tipus text la considerarà String. Si conté true o false la considerarà booleana.

El nom d'una variable haurà d'estar format per lletres, nombres, guions baixos o símbols dòlar ($),

Una variable s'inicialitza quan s'estableix el seu contingut o valor per primera vegada.

Una variable es pot declarar i inicialitzar al mateix temps. Per exemple podríem escriure var num = 30; amb el que la variable ha estat declarada i inicialitzada en una mateixa línia.

Javascript distingeix entre majúscules i minúscules .

Javascript no requereix declaració del tipus de les variables, i fins i tot permet que una variable emmagatzemi contingut de diferents tipus en diferents moments. Per exemple podríem usar mun = 30; i en un lloc posterior escriure mun = 'es menor'
