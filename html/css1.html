CSS

CSS significa "Cascading Style Sheets" o el que és el mateix Fulls d'Estil en Cascada. És un llenguatge de fulls d'estil utilitzat per descriure la semàntica de presentació (l'aspecte i format) d'un document escrit en un llenguatge de marques. La seva aplicació més comuna és dissenyar pàgines web escrites en HTML i XHTML, però el llenguatge també pot ser aplicat a qualsevol classe de document XML, incloent-hi SVG i XUL.

Per saber més pots visitar aquesta web.

Els estils en CSS es poden utilitzar dins d'una pàgina HTML o en un fitxer a part:

  • En cas d'utilitzar CSS dins d'una mateixa pàgina HTMl haurem d'escriure el codi a la capçalera de la mateixa pàgina HTML, dins de les etiquetes <head></head>. En aquest cas escriurem les eiquetes <style> </style> i l'estil dins d'aquestes etiquetes.
  • Es pot utilitzar un arxiu diferent per fer el full d'estils. En aquest cas crearem un arxiu amb l'extensió CSS i el cridarem desde HTML entre les etiquetes <head></head> amb el següent codi, essent "estil-web.css" l'arxiu CSS amb els estils d'aquesta pàgina HTML.

<link rel="stylesheet" href="estil-web.css">

Una regla CSS está formada sempre per dues parts, els selectors i la declaració:

Els selectors
Indica a quí s'ha d'executar la declaració
La declaració
Indica qué s'ha de fer i sempre es troba entre claus({}). Cada declaració ha d'estar separada per un punt i coma (;)
