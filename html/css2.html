Declaracions de CSS més utilitzades

Els colors

Els colors a CSS es poden posar de diferents formes:

  • Amb el nom del color (en anglès), per exemple: 
    h1{
color: red;
}
  • Amb els valors de codi RGB (Red, Green and Blue) amb els valors, pels 3 colors bàsics, des de 0 fins a 255, per exemple (el color vermell): 
    h1{
color: rgb(255,0,0);
}
    es poden utilitzar percentatges, com per exemple 
    h1{
color: rgb(100%,0%,0%);
}
    i també amplian els colors amb la opacitat, possant el codi rgba, on el últim valor representa la opacitat (1 per opacitat total i 0 per color transparent) per exemple. 
    h1{
color: rgba(255,0,0,0.5);
}
  • Amb el codi hexadecimal dels colors primaris (vermell, verd i blau). S'ecriu amb un coixinet (#) davant del codi seguit de la combinació de dos caracters del codi hexadecimal (del 0 al 9 i de la A fins a la F) per a cada color, com per exemple (el color vermell): 
    h1{
color: #FF0000;
}
