CSS

Els selectors

Els selector son els elements als qui s'aplicará l'estil, o la declaració de la regla CSS. Podem trobar diferents tipus de selectors: selectors universals, de tipus o etiqueta, de clase, de ID, ...

Depenent del tipus de selector que utilitzem els cridarem d'una manera o d'una altra en el CSS.

  • Per a selectors de tipus universal utilitzem com a selector l'asterisc (*). Així si volem cambiar tot el color de la pàgina de color gris utilitzarem el següent codi.
*{
color:gray;
}
  • Per a selectors de tipus o etiqueta es criden mitjançant la etiqueta. Així si volem cambiar el format del contingut dels elements inclosos en un <h1> o farem de la següent manera. 
    h1{
font-size:2em;
}
    Així canviariem totes les fonts compreses entre etiquetes <h1> al doble del seu tamany original.
    Si per exemple volem canviar el color de fons, per el color vermell, de tots els paràgrafs ho farem de la següent manera. 
    p{
background-color:red;
}
  • Per a selectors de tipus classe, primer s'ha d'aplicar la classe al lloc on la volem. Pot ser un paràgraf, una etiqueta DIV, una etiqueta SPAN o qualsevol altra, de la següent manera <span class="name">. Per cridar-lo en el CSS utilitzarem el punt, seguit del nom de la classe. Així si volem canviar el color de la lletra de la classe "name", escriurem ... 
    .name{
color:white;
}
  • També podem identificar els selectors amb un ID. El primer de tot serà aplicar l'identificador al lloc on el volem. Pot ser un paràgraf, una etiqueta DIV, una etiqueta SPAN o qualsevol altra, de la següent manera <div id="p1">. Per cridar-lo en el CSS utilitzarem el coixinet (#), seguit del nom del identificador. Així si volem augmentar 10 pixels els marges a "p1", escriurem ... 
    #p1{
margin:10px;
}
  • Podem fer combinacions de selectors. Deixant un espai entre selectors seleccinem totes les classes o identificadors d'una etiqueta concreta. Per exemple si volem canviar el format de las classe "em" de les etiquetes "h1" ho farem de la següent forma ... 
    h1 em{
font-size: 1.5em;
font-weight: bold;
}
    ...o també sense espais... 
    em#p1{
color: gray;
font-weight: bold;
}

    Podeu veure una guia ràpida dels selectors i les declaracions en aquests enllaç
