CSS

Altres selectors (les pseudoclasses)

A CSS tenim altres selectors més especifics, les pseudoclasses, que permeten fer canvis de estil en elements concrets de la pàgina. Aquestes paseudoclasses són cridades per el CSS amb dos punts verticals (:). Els més utilitzats són:

  • :link ---> format per enllaç no visitat.
  • :visited ---> format per enllaç visitat.
  • :hover ---> format al passar el ratolí per sobre.
  • :active ---> format per enllaç actiu.

Exemples:

a:link {
    color: #FF0000;
}

a:visited {
    color: #00FF00;
}

a:hover {
    color: #FF00FF;
}

a:active {
    color: #0000FF;
}

Selectors per a llistes

Les llistes ordenades i desordenades utilitzades amb les etiquetes <ul> i <ol> d'HTML es poden canviar amb els estils de CSS. Podem utilitzar la declaració "list-style" per canviar alguns aspectes de les llistes. Per exemple:

ul{list-style-type: circle;}
ul{list-style-type: square;}
ol{list-style-type: upper-roman;}
ol{list-style-type: lower-alpha;}

es veurà de la següent manera...

Selectors per a taules

Les taules també poden asociar-se a diferents estils, molts dels quals són semblants als estils de text. Encara que hi han estils només per a taules. Aquests són els més utilizats.
