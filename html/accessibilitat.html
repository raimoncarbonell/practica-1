ACCESSIBILITAT

Què és l'Accessibilitat Web?

L'accessibilitat Web significa que persones amb algun tipus de discapacitat van a poder fer ús de la Web. En concret, en parlar d'accessibilitat Web s'està fent referència a un disseny Web que va a permetre que aquestes persones puguin percebre, entendre, navegar i interactuar amb la Web, aportant al seu torn continguts. L'accessibilitat Web també beneficia a altres persones, incloent persones d'edat avançada que han vist minvades les seves habilitat a conseqüència de l'edat.

L'accessibilitat Web engloba molts tipus de discapacitats, incloent problemes visuals, auditius, físics, cognitius, neurològics i del parla. El document "Com utilitzen la Web persones amb discapacitat" mostra la forma en la qual diferents discapacitats poden dificultar la utilització de la Web i inclou alguns escenaris de persones amb discapacitat utilitzant la Web.

Aquesta flexibilitat beneficiarà a totes aquelles persones que utilitzen la Web, incloent persones que no tenen cap discapacitat però que, a causa de determinades situacions, tenen dificultats per accedir a la Web (per exemple, una connexió lenta), també estaríem parlant d'aquelles persones que sofreixen una incapacitat transitòria (per exemple, un braç trencat), i de persones d'edat avançada.


Pautes d'Accessibilitat al Contingut de la Web.

En resum:
  1. Imatges i animacions: Utilitzarem l’atribut alt per a descriure la funció de cada element visual.
  2. Mapes d’imatge: Utilitzarem l’element map de tipus client i text per a les zones actives.
  3. Multimèdia: Proporcionarem subtítols i transcripció del so, i descripció del vídeo.
  4. Enllaços hipertextuals: Utilitzarem text que tingui sentit llegit fora de context. Per exemple, evitem "pinxa aquí".
  5. Organització de les pàgines: Utilitzarem encapçalats, llistes i estructura consistent. Utilitzem CSS per a la maquetació on sigui possible.
  6. Gràfics i esquemes: Resumirem o utilitzarem l’atribut longdesc.
  7. Scripts, applets i plug-ins: Oferirem contingut alternatiu si les funcions noves no son accessibles.
  8. Marcs (Frames): Utilitzi l’element noframes i títols amb sentit.
  9. Taules: Disposarem que puguin llegir-se línia a línia. Resumeirem.
  10. Revisarem el nostre treball: Verificarem. Utilitzarem les eines, llista de comprovació i pautes de www.w3.org/TR/WCAG.

Quines eines ens poden ajudar per saber si la meva web compleix el nivell AA?

Existeixen eines automàtiques per l’avaluació de l’accessibilitat que permeten identificar alguns problemes d’accessibilitat del codi. Entre elles estan:
  • TAW (Test de Accesibilidad Web) : http://www.tawdis.net/
  • eXaminator : http://examinator.ws/
