ACCÉS A ELEMENTS DE DOCUMENTS (DOM)

El model de objectes de DOM es una interfície (connexió entre el codi HTML i el codi de programació )per poder accedir i modificar tant en estructura i el estil els codi HTML de la pagina web. Permet connectar el codi HTML amb el llenguatge de programació. Permetre detectar ,actuar i actuar sobre el codi HTML.

Control del temps de crida a funcions

El mètode setInterval es un mètode que permet cridar o evaluar una expressió en un intervals especificats (en mil·lisegons). El mètode setInterval() continuarà les crides a la funció fins que s’executi la clearInterval() Per executar una funció només una vegada, després d'un número especificat de mil·lisegons, ús el setTimeout() mètode.
